üìö Usage Guidelines

When to Use Each Color

  • Primary Purple (#8B5CF6): Main CTAs, primary buttons, important links
  • Secondary Purple (#A78BFA): Secondary actions, hover states
  • Light Purple (#C4B5FD): Backgrounds, subtle highlights
  • Accent Pink (#EC4899): Special promotions, featured items, premium badges
  • Dark (#1F2937): Main headings, body text
  • Gray Dark (#4B5563): Body text, descriptions
  • Gray Medium (#9CA3AF): Secondary text, captions

Typography Best Practices

  • H1 (48px): Use only once per page for main page title
  • H2 (36px): Section headers, major content divisions
  • H3 (28px): Subsection headers, card titles
  • Body Large (18px): Hero descriptions, important announcements
  • Body Regular (16px): Default body text, descriptions
  • Body Small (14px): Secondary information, labels
  • Caption (12px): Timestamps, footnotes, legal text
